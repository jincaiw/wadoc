# 删除 Microsoft AutoUpdate

在安装 Office 之后，Mac 电脑总会自动弹出 Microsoft Auto Update，以下是关闭方法。

![Microsoft AutoUpdate 弹窗](https://raw.githubusercontent.com/jincaiw/pic/master/uPic/image-20230227145643024.png)

## 删除方法

1. 打开**访达**，顶部菜单选择 **前往 → 前往文件夹**

2. 输入以下路径，点击**前往**：

   ```
   /Library/Application Support/Microsoft/
   ```

3. 删除文件夹 `MAU2.0` 即可。
