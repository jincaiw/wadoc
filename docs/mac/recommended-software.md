# Mac 常用软件推荐

## 常用利器

### IINA

免费开源的影音播放器，UI 简洁美观，支持画中画，Mac 平台最好的免费播放器之一。

### Free Download Manager（FDM）

Mac 平台免费的下载管理工具，支持浏览器插件。

### Picture View

优秀的 Mac 本地看图应用，加载图片快速，免费使用，可替代默认看图软件。

### 腾讯柠檬清理

Mac OS 平台免费的系统清理工具。

### ClashX Pro

Mac OS 必备利器，网络代理工具。

## 效率工具

### Bob

macOS 平台翻译和 OCR 软件，可快速划词翻译、截图 OCR。

### Rectangle

免费开源的分屏软件，支持键盘快捷键和贴靠区域。

### Mac Mouse Fix

开源免费的工具，用于在 macOS 上优化第三方鼠标。

## 转换工具

### ezip

Mac 免费压缩软件，UI 美观，支持 7z、zip、rar 等格式。

### HandBrake

开源的视频格式转换压缩工具。

### 洋芋田图像工具箱

图片加水印、长图拼接、图片裁剪分割、富文本制图、图片压缩、格式转换等。

## 虚拟机

### Parallels Desktop

Mac 平台最好用的虚拟机工具。

### 网易 MuMu

Mac 平台的安卓模拟器。

## 办公软件

### ToDesk

免费远程协助软件，国内 Mac 用户远程协助的最优选择。

### WPS for Mac

免费国产办公软件，满足日常办公需求。

### Visual Studio Code

微软出品的编辑器，Mac 最强编辑器。Extensions 中搜索 `Chinese` 可安装中文语言包。

### Feem

局域网文件传输神器，支持 iOS、Android、Windows、macOS、Linux 之间互传文件。
