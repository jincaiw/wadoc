# 基础概念

本文档介绍本站使用的一些核心概念。

## 文档即代码

> Docs as Code

将文档视为代码来管理，使用版本控制、持续集成、代码评审等软件开发实践来管理文档。

### 核心理念

- **版本控制**：所有文档通过 Git 管理，历史可追溯
- **纯文本**：使用 Markdown 编写，专注于内容而非样式
- **自动化**：通过 CI/CD 自动构建和部署
- **协作**：像管理代码一样管理文档变更
