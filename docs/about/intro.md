# 项目介绍

## 关于

**王记文档** 是一个个人知识库与技术文档站点。

使用 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 构建，
致力于提供清晰、高效、易于维护的文档体验。

## 技术栈

- **构建工具**：[MkDocs](https://www.mkdocs.org/)
- **主题**：[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)
- **部署**：GitHub Pages + GitHub Actions
- **代码托管**：[GitHub](https://github.com/jincaw/wadoc)
- **自定义域名**：`docx.mujizi.com`

## 写作风格

- 使用简洁清晰的中文
- 技术术语保留英文原文
- 中英文之间加空格
- 代码块指定语言标识
- 每页只保留一个一级标题
